# Claude Code Guidelines for dertown

## Debugging & Investigation

When asked to diagnose, investigate, or examine an issue:
- **Always invoke the debugging skill first** before exploring code or systems
- The debugging skill provides a structured approach that prevents tangential investigation
- Don't assume the problem is in a particular area; let the methodology guide you to the root cause
- This prevents wasted time investigating adjacent systems that seem related but aren't the actual issue
